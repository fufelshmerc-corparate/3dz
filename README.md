## Ассемблер для Учебной Виртуальной Машины (УВМ)
Описание
Ассемблер для учебной виртуальной машины (УВМ) преобразует текстовое представление программ в бинарный формат, понятный виртуальной машине.

## Синтаксис языка ассемблера

Общий формат команд:
[метка:]

мнемоника [аргумент1, аргумент2, ...] ; комментарий

## Поддерживаемые команды:

### CONST - Загрузка константы
const значение
Размер: 4 байта
Загружает константу на стек
Пример: const 291 или const 0x123

### LOAD - Чтение значения из памяти
load
Размер: 1 байт
Читает значение из памяти по адресу с вершины стека

### STORE - Запись значения в память
store смещение
Размер: 2 байта
Записывает значение в память по адресу (адрес + смещение)

### SHR - Побитовый арифметический сдвиг вправо
shr смещение
Размер: 2 байта
Выполняет сдвиг вправо

## Форматы чисел:

Десятичные: 123

Шестнадцатеричные: 0x7B

Двоичные: 0b1101

Метки:
метка:
const значение
Метки можно использовать как аргументы команд.

## Комментарии:
Комментарии начинаются с ; и продолжаются до конца строки.

## Пример программы:
```; Простая программа
start:
const 0x100
load
store 10
shr 20```

## Использование ассемблера:

Обычный режим
python asm.py input.asm output.bin

Режим тестирования
python asm.py input.asm output.bin --test

Внутреннее представление:
В режиме тестирования отображается:

Код операции (Opcode)

Операнд (Operand)

Бинарное представление

размер файла
